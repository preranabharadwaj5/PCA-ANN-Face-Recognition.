# PCA-ANN Face Recognition System

## Overview

This project implements a Face Recognition System using Principal Component Analysis (PCA) for feature extraction and Artificial Neural Networks (ANN) for face classification.

The PCA algorithm reduces image dimensionality by extracting Eigenfaces, while ANN is used to classify faces based on extracted features.

## Features

- Face Recognition
- PCA Feature Extraction
- Eigenface Generation
- ANN Classification
- Dimensionality Reduction

## Technologies Used

- Python
- NumPy
- OpenCV
- Scikit-Learn
- TensorFlow/Keras
- Jupyter Notebook

## Dataset

The dataset consists of face images of multiple individuals stored in separate folders.

## Workflow
